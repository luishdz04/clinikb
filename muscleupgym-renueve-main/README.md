# muscleupgym-renueve

Proyecto inicializado con Next.js (App Router) y TypeScript.

## Requisitos
- Node.js >= 18 (verificado: contenedor tiene Node 22)
- npm (incluido con Node)

## Scripts
- `npm run dev` — Arranca servidor de desarrollo en `http://localhost:3000`
- `npm run build` — Compila la aplicación para producción
- `npm start` — Inicia la versión compilada
- `npm run lint` — Linter de código

## Instalación
```bash
npm install
```

## Uso rápido
```bash
npm run dev
```
Abrir luego `http://localhost:3000`.
