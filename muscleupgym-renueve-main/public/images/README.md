# Imágenes

Esta carpeta contiene todas las imágenes del sitio organizadas por categorías.

## Subcarpetas:

### `/heroes`
Imágenes hero para secciones principales (banner, páginas, etc.)

### `/gallery`
Fotos del gimnasio, equipo, instalaciones, eventos, etc.

### `/icons`
Iconos e ilustraciones personalizadas en formato SVG/PNG
